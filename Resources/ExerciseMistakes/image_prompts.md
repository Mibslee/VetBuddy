# VetBuddy Common Mistake Image Prompts

Use the locally authorized face reference photo supplied by the app owner as the identity reference. Do not commit the reference photo, local path, or generated face-bearing images to the public source repository.

The model must resemble the reference photo: Chinese male, short black hair, rectangular rimless or light metal glasses, fair skin, calm neutral expression, similar face shape and facial proportions. Generate the rest of the body as an average-build middle-aged man in clean rehab-friendly sportswear. Keep clothes, lighting, room background, floor and camera angle consistent with the correct action sequence where possible. Use a clean indoor home setting, no medical branding.

For every output: 4:3 PNG, clear full-body composition, leave margin for app cropping, show an obvious red X or subtle red warning marker only if it does not cover the body.
