# Privacy Statement

Last updated: 2026-06-15

VetBuddy / 老铁 is designed as a local-first wellness app. This repository contains source code only and does not include personal voice samples, generated cloned voices, user photos, App Store screenshots, local device data, or other private materials.

## Data Processed by the App

Depending on user actions and permissions, the app may process the following data on device:

- Health assessment answers, such as age range, activity level, and self-reported health conditions.
- Training records, such as completed exercises, duration, and check-in history.
- Diet records, such as food name, portion size, and estimated macronutrients.
- Apple Health data, after user authorization, including step count, heart rate, and body mass.
- Camera frames for local movement or pose guidance, when camera permission is granted.
- Generated poster images, when the user chooses to save or share them.

## Local Processing

The current source code is designed to store app records locally on the user's device. It does not include backend upload code, account login, advertising tracking, or third-party analytics integration.

Apple Health data is read only after explicit user authorization and is used for local display and local nutrition target estimation.

Camera frames are intended for local motion guidance. They are not uploaded by the app.

## Data Sharing

The app source code does not contain logic to sell, upload, or share user health data with third parties.

If future versions add cloud sync, analytics, crash reporting, accounts, or third-party SDKs, this privacy statement and App Store privacy disclosures must be updated before release.

## Excluded Private Materials

The following materials must not be committed to this repository:

- Raw voice samples
- Cloned voice output files
- Personal photos or likeness-based generated media
- App Store submission screenshots or archives
- Local build artifacts
- API keys, tokens, passwords, certificates, provisioning profiles, or other secrets

## Medical Disclaimer

VetBuddy is intended for wellness, exercise guidance, and diet logging. It does not provide medical diagnosis, treatment, prescription, or professional medical advice. Users with pain, chronic disease, recent surgery, fall risk, or other health concerns should consult a qualified professional.
