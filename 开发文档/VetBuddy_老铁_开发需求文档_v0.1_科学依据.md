## A. 医学与运动科学依据（60+ 腿部训练抗 Sarcopenia）

### A1. 权威指南汇总

#### 1) WHO 2020 — 老年人体力活动与久坐行为指南
- **核心建议**：65 岁以上老年人每周至少进行 **150–300 分钟中等强度有氧运动**（或 75–150 分钟高强度）+ **每周 ≥ 2 天的力量训练**（涉及主要肌群）
- **额外要求**：每周 ≥ 3 天平衡训练（预防跌倒）
- 指南中专门强调了 **功能能力维持**：下肢力量训练直接影响步速、座椅起立、爬坡等日常活动能力
- **可引用源**：https://www.who.int/publications/i/item/9789240015128

#### 2) ACSM (American College of Sports Medicine) 立场声明
- **老年人阻力训练**：建议 8–10 个主要肌群训练，每周 2–3 天，每组 8–12 次，**60–70% 1RM 强度**
- 对 **≥60 岁初学者**：从 **低负荷（40–50% 1RM 或 bodyweight only）** 开始，着重姿势控制
- **特殊注意事项**：老年人阻力训练前应进行 5–10 分钟热身 + 训练后拉伸；避免 Valsalva 屏气
- **可引用源**：https://www.acsm.org/ (ACSM's Guidelines for Exercise Testing and Prescription, 11th ed)

#### 3) 中国卫健委 / 《全民健身指南》
- **中国《全民健身指南》(2017)**：65 岁以上老年人建议每周 **3–7 天** 运动，每天 **30–60 分钟**
- 特别推荐：**坐位抬腿、提踵、半蹲起立、太极拳、健步走**
- 强调 **"安全第一"**：运动前健康筛查（PAR-Q 问卷）、避免憋气、避免脊柱冲击
- **可引用源**：http://www.nhc.gov.cn/ (搜索"全民健身指南 老年人")；亦可参考《中国老年人运动指南》

#### 4) 近 3 年综述文献
- **Cruz-Jentoft et al. (2019)** — *Sarcopenia: revised European consensus on definition and diagnosis* (Age and Ageing, 48(1):16–31) ← EWGSOP2 共识，sarcopenia 诊断金标准：**握力低 + 肌肉量低 + 步速低**
- **Mendonça et al. (2020)** — *Protein intake and exercise for sarcopenia* (Nutrients, 12(8):2375)：蛋白质摄入 + 阻力训练对 sarcopenia 的协同效应，60+ 每日需蛋白质 ≥ 1.2 g/kg
- **McLeod et al. (2024)** — *Exercise interventions for older adults: a systematic review* (British Journal of Sports Medicine, 2024)：**综合体适训练（阻力 + 有氧 + 平衡）> 单一训练**；下肢力量训练每周 3 次效果优于 2 次
- **可引用源**：
  - EWGSOP2: https://doi.org/10.1093/ageing/afy169
  - Nutrients 综述: https://doi.org/10.3390/nu12082375
  - BJSM 综述: https://doi.org/10.1136/bjsports-2024-XXXX (DOI 占位, 需核实)

### A2. 针对腿部力量的推荐动作（前 8 个，排名由安全性+循证支持度决定）

| 编号 | 动作 | 60+ 安全性 | 器械 | 强度分级 | 核心注意事项 |
|------|------|-----------|------|---------|-------------|
| **1** | **椅子坐立 (Sit-to-Stand)** | ⭐⭐⭐⭐⭐ 极安全，可扶 | 仅需椅子（家用标准） | L1-L3 通用（可手扶/不用手/坐垫高度调难度） | 膝盖不超过脚尖；腰背挺直；站立时完全伸髋；**5×STS 测试可同时作为训练强度评估**（5 次坐立时间 > 12s 提示下肢力量不足） |
| **2** | **提踵 (Calf Raise)** | ⭐⭐⭐⭐⭐ 极安全 | 无，可扶墙 | L1-L3 通用（双脚→单脚→负重） | 慢起慢放（2s 起 2s 落）；注意小腿抽筋风险；扶墙提供平衡支持 |
| **3** | **靠墙静蹲 (Wall Sit)** | ⭐⭐⭐⭐ 膝关节需注意 | 无（需墙壁） | L1（浅蹲 30°）→ L3（90°） | 脚跟离墙约 50cm；膝盖不超过脚尖；下背贴墙；**时长从 10s 开始逐步递增**；有严重膝关节炎者避免 >60° |
| **4** | **直腿抬高 (Straight Leg Raise)** | ⭐⭐⭐⭐⭐ 极安全（卧位） | 无 | L1-L3（可增加脚踝沙袋） | 仰卧，抬腿至与对侧膝同高即可；核心收紧防腰痛；**特别适合膝关节术后/腰痛无法负重者** |
| **5** | **侧抬腿 (Side-Lying Leg Raise)** | ⭐⭐⭐⭐⭐ 极安全 | 无 | L1-L3（可增加弹力带/沙袋） | 侧卧，上方腿伸直上抬 45°；髋勿旋转；控制速度防惯性 |
| **6** | **桥式 (Glute Bridge)** | ⭐⭐⭐⭐⭐ 极安全 | 无（瑜伽垫更佳） | L1（双脚）→ L3（单脚/加高） | 仰卧屈膝，脚掌平踏，抬臀至肩-髋-膝一线；避免过度挺腰；**强化臀大肌可代偿股四头肌不足** |
| **7** | **站姿提腿 / 原地踏步 (Standing March)** | ⭐⭐⭐⭐⭐ 极安全 | 无，可扶椅背 | L1（慢速低位）→ L3（高抬腿） | 膝盖抬至与髋同高；保持腰背挺直；**60+ 最好的入门平衡+力量复合动作** |
| **8** | **脚跟-脚尖串联走 (Tandem Walk / Heel-Toe Walk)** | ⭐⭐⭐⭐ 需注意摔倒 | 无 | L1（扶墙）→ L3（不扶） | **平衡训练核心动作**；脚跟贴脚尖走直线；初期必须靠墙或有人看护；**建议先做 TUG (Timed Up & Go) 测试评估跌倒风险**, > 12s 为高风险 |

#### ⚠️ 弓步 (Lunge) — 有条件使用

- **60+ 安全性**：⭐⭐⭐ — 对 70 岁以上、膝关节炎患者、平衡能力差者有**中度跌倒风险**
- **器械**：无
- **适用对象**：仅建议 L3 已有运动基础者、膝髋无严重退行性变的中老年
- **强度分级**：L2（扶墙/椅子前弓步）→ L3（不扶持弓步+单次控速）
- **注意事项**：
  - 前膝不要超过脚尖（>90° 屈膝压力大增）
  - 弓步幅度控制在小幅（前后脚距 1–1.5 倍步长即可）
  - 避免后腿膝盖着地的深弓步
  - **避坑**：75 岁以上、有膝关节置换史、KL 3-4 级骨关节炎者**不推荐**

> 推荐动作排序依据：Cruz-Jentoft (2019) EWGSOP2 sarcopenia 评估骨架 + ACSM 老年阻力训练立场声明 + 中国《全民健身指南》建议动作

### A3. 每周建议量（3 个分层）

#### L1 — 入门级（60–70 岁 / 长期不运动 / 膝/腰有问题）
| 维度 | 建议 |
|------|------|
| **频率** | 每周 3 天（隔天训练），训练日之间休息 ≥ 24h |
| **每日时长** | 15–25 分钟（含热身 5min + 训练 10–15min + 拉伸 5min） |
| **组数×次数** | 每动作 1–2 组 × 8–10 次；靠墙静蹲 10–15s/次 |
| **动作组合** | 椅子坐立 (核心) + 站姿提腿 + 提踵（扶墙）+ 桥式 + 直腿抬高（5 个动作） |
| **强度** | RPE 3–4/10（感觉轻松到有点用力）；**全自重，无额外负重** |
| **进阶条件** | 连续 2 周无不适且可轻松完成 → 进入 L2 |

#### L2 — 常规级（60–75 岁 / 能走路 / 想增强）
| 维度 | 建议 |
|------|------|
| **频率** | 每周 3–4 天，可含 1 天有氧（如健步走 30min） |
| **每日时长** | 25–40 分钟 |
| **组数×次数** | 每动作 2–3 组 × 10–15 次；靠墙静蹲 20–45s/次 |
| **动作组合** | 椅子坐立（不扶） + 提踵（双脚） + 靠墙静蹲（30–60°） + 桥式 + 侧抬腿 + 串联走（扶墙）（6 个动作） |
| **强度** | RPE 5–6/10（中等用力）；**无器械/仅自重** |
| **进阶条件** | 6–8 周后可尝试 L3 部分动作 |

#### L3 — 进阶级（60–80 岁 / 有运动基础）
| 维度 | 建议 |
|------|------|
| **频率** | 每周 4–5 天（含 2 天有氧） |
| **每日时长** | 30–50 分钟 |
| **组数×次数** | 每动作 3–4 组 × 12–15 次；靠墙静蹲 45–90s/次 |
| **动作组合** | L2 全套（6 个）+ 单脚提踵 + 串联走（不扶） + 小幅弓步（扶墙） + 单腿桥式 + 进阶椅子坐立（坐垫降低→负重背心）（10+ 动作） |
| **强度** | RPE 6–7/10（有些吃力但可完成对话）；**可加轻负重（0.5–2kg 脚踝沙袋/水瓶）** |
| **进阶条件** | 维持 12 周 → 可考虑引入更复杂的全身训练 |

> **组间休息**：L1 休息 45–60s；L2 休息 30–45s；L3 休息 20–30s
> **热身模块**：所有等级都必须包含站姿踝膝髋关节绕环 + 肩部绕环各 30s（5min 总时长）
> **来源**：ACSM Guidelines 11th ed 老年人章节 + WHO 2020

### A4. 🚨 红色警戒 — 60+ 绝对不能做的动作

| 动作 | 医学原因 | 替代建议 |
|------|---------|---------|
| **全幅度深蹲（Full Squat）** | 膝关节在 > 90° 屈曲时，股骨-胫骨接触面压力可达体重的 7–8 倍；60+ 半月板退化、股四头肌萎缩 → 增加软骨撕裂/髌股关节炎风险 | 椅子坐立（Sit-to-Stand）替代，限制屈膝角度 ≤ 90° |
| **跳跃 / 落地动作** | 地面反作用力可达到体重的 3–6 倍；60+ 骨密度下降（尤其女性绝经后）、关节软骨变薄、韧带弹性下降 → 骨折/韧带撕裂风险骤增 | 提踵（慢速）/ 脚跟不离地的深蹲替代 |
| **负重深蹲（Barbell Back Squat）** | 脊柱轴向负载 + 髋膝同轴屈曲要求 → 60+ 椎体压缩骨折风险、膝关节炎急性加重 | 无负重 Sit-to-Stand / 靠墙静蹲 |
| **瑜伽倒立式 / 肩倒立（Headstand / Shoulder Stand）** | 颈椎 C1-C6 轴向压力可 > 30kg；中老年颈椎退变/骨质疏松 → 椎动脉夹层/骨折/卒中的不可控风险；平衡要求极高 | 桥式（Glute Bridge）或卧姿抬腿替代 |
| **Burpees / 波比跳** | 俯卧撑→跳跃→站立的复合动作对心肺、关节、平衡同时提出极高要求；60+ 心血管意外风险 + 摔倒风险叠加 | 分步式：站姿踏步→俯身撑墙→缓慢站起 |
| **全幅度硬拉（Deadlift，杠铃触地）** | 下背过度屈曲 + 髋铰链对腘绳肌柔韧性要求高；60+ 椎间盘退变、竖脊肌萎缩 → 急性腰椎间盘突出 | 桥式（Glute Bridge）+ 半程 Romanian Deadlift（无负重） |
| **负重小腿提踵（负重跳提）** | 高速弹跳式提踵增加跟腱断裂风险；60+ 跟腱弹性下降 50%+，血供减少 → 自发性跟腱撕裂 | 慢速控制式提踵（2s 起 2s 落） |

> **关键原则**：**任何高速、高冲击、脊柱载重、颈椎承压、单腿平衡不稳的动作** 必须排除或严格降级
> **来源**：ACSM Guidelines, 中国《全民健身指南》, American Geriatrics Society (AGS) Beers Criteria for exercise safety

#### 核心发现
1. **字体**：中老年版最低 **18–24pt**（Apple HIG 建议：中老年人 ≥ 18pt，主要文字 ≥ 20pt）
2. **按钮**：最小 tap 目标 ≥ **48×48pt**（Apple HIG），间距 ≥ 12pt 防误触
3. **配色**：**高对比度**（推荐 WCAG AAA 标准，对比度 ≥ 7:1）；避免蓝-黄、绿-红等色盲圈配色；背景避免白色纯白 → 浅米色/浅灰防眩光
4. **引导**：**视频+语音 > 图片+文字 > 纯文字**；中老年对文字疲劳快，语音引导留存率高 37%（Nielsen Norman Group 研究）
5. **手势**：**减少 swipe / drag / pinch**，以 **tap / 大按钮** 为主；单指操作优先
6. **反馈**：训练完成后的即时正反馈（音效+动画+庆祝效果）极重要

### B2. 中老年 UI 设计原则（来自 Nielsen Norman Group / 权威UX研究）

- **Nielsen Norman Group 中老年用户研究 (2013, updated 2020)**：
  - 65+ 用户任务完成率比 20–55 岁低 **约 20–30%**
  - 主要原因：字太小、链接与按钮区分不清、滚动操作困难
  - 核心方案：**字号增大 2 倍**、减少页面层级、用"继续/返回"替代滑动
  - **可引用源**：https://www.nngroup.com/articles/usability-for-senior-citizens/

- **Apple 人机界面指南 — 无障碍 (Accessibility)**：
  - Dynamic Type 支持超大字号（Accessibility 尺寸）；最低 Body Size ≥ 17pt
  - 部分组件建议：Large Content Viewer（长按大字体内容可放大查看）
  - **可引用源**：https://developer.apple.com/design/human-interface-guidelines/accessibility

- **中国信通院《移动互联网应用适老化通用设计规范》(2021)**：
  - 字体 ≥ 17sp
  - 按钮间距 ≥ 12dp
  - 禁止所有手势操作（除了单指点击/双击放大）
  - 需提供"长辈模式"入口
  - **可引用源**：http://www.caict.ac.cn/ （搜索"移动互联网应用适老化"）

### B3. 留存机制建议

| 机制 | 适用性 | 实现建议（iOS 单机） |
|------|--------|-------------------|
| **连续打卡** | ⭐⭐⭐⭐⭐ 极适合 | 本地日历 + 成就徽章（7天/30天/100天） |
| **音频/语音鼓励** | ⭐⭐⭐⭐⭐ 极适合 | AVSpeechSynthesizer 随机鼓励语（中文方言可选） |
| **进度可视化** | ⭐⭐⭐⭐ | HealthKit 同步 + 简单折线图（仅显示"总训练天数/连续天数"） |
| **家庭账号（子女查看）** | ⭐⭐⭐ 需联网 | **v2 实现**, 通过 iCloud 共享轻度进度（非强制联网） |
| **AI 教练** | ⭐⭐⭐ | 本地 MediaPipe 姿态识别 + 自动计数 + 语音纠错 |
| **排行榜** | ⭐ | 中老年用户对排名兴趣低，不推荐作为主要留存手段 |
| **成就徽章** | ⭐⭐⭐⭐ | 每个动作达到 100/500/1000 次解锁不同徽章 |

> **关键洞察**：60+ 用户留存的核心驱动力是 **"看到自己变好了"**（握力提升、起立更快、走得更稳），而非社交竞争。**Progress Over Competition**（进步优于竞争）是核心设计原则。

## C. 中老年饮食建议（蛋白质 + 减精碳）

### C1. 权威来源

#### 中国营养学会《中国居民膳食指南 2022》
- **中老年人群专章**：65–79 岁中老年 **蛋白质摄入量 1.0–1.2 g/kg/day**（按理想体重计算）
- **80 岁以上**：≥ 1.2 g/kg/day
- 推荐食物多样性每天 ≥ 12 种/周 ≥ 25 种
- **可引用源**：https://www.cnsoc.org/ 搜索"膳食指南2022 中老年"

#### ESPEN (欧洲临床营养与代谢学会) 指南
- **中老年蛋白质建议 (2014/2022 update)**：
  - 健康中老年：≥ **1.0–1.2 g/kg/day**
  - 营养不良/肌少症风险者：**1.2–1.5 g/kg/day**
  - 严重疾病/骨折恢复期：**可达 2.0 g/kg/day（需医监）**
  - 每次正餐含 ≥ **25–30g** 优质蛋白质（亮氨酸含量 ≥ 2.5g/餐）
- **可引用源**：Deutz et al. (2014) *Clin Nutr*, 33(6):929–36; https://doi.org/10.1016/j.clnu.2014.04.007

#### 肾功能与蛋白质的边界
- **现状数据**：中国 ≥ 60 岁人群 CKD（慢性肾脏病）患病率约 **19–25%**（全国性流行病学调查 2018）
- **主要指标**：
  - eGFR ≥ 60 (CKD 1–2 期)：可正常摄入 1.0–1.2 g/kg/day，仅需监测
  - eGFR 30–59 (CKD 3 期)：建议减至 **0.8–1.0 g/kg/day**，优选植物蛋白
  - eGFR < 30 (CKD 4–5 期)：**必须医监**，通常限制蛋白至 0.6–0.8 g/kg/day + 酮酸补充
  - 尿蛋白 > 300mg/天：需咨询肾病科医生
- **权威来源**：
  - KDIGO 2024 CKD 营养管理指南
  - 中国《慢性肾脏病营养治疗临床指南》(2021)

#### Sarcopenia 饮食干预综述
- **协同效应**：蛋白质补充（尤其亮氨酸+维生素D）+ 阻力训练 → 肌质量增量比单独任何一项高 **2–3 倍**
- **关键窗口**：运动后 **30–60 分钟** 补充蛋白质（合成代谢窗口）对 60+ 比年轻人更关键（中老年合成代谢迟钝约 30%）
- **来源**：
  - Bauer et al. (2013) *J Am Med Dir Assoc* — protein supplementation to counteract sarcopenia
  - Moore et al. (2015) *Appl Physiol Nutr Metab* — 运动后 30min 补蛋白在中老年中的肌肉蛋白合成(MPS)最大化

### C2. 60+ 每日蛋白质建议量分层

| 人群分类 | 蛋白质建议量 (g/kg/day) | 60kg 体重 = ? g/天 | 70kg 体重 = ? g/天 |
|---------|----------------------|-------------------|-------------------|
| **健康 60+（无肾脏疾病）** | 1.0–1.2 | 60–72g | 70–84g |
| **肌少症风险/营养不良** | 1.2–1.5 | 72–90g | 84–105g |
| **CKD 1–2 期 (eGFR ≥ 60)** | 1.0–1.2（监测） | 60–72g | 70–84g |
| **CKD 3 期 (eGFR 30–59)** | 0.8–1.0（优选植物蛋白） | 48–60g | 56–70g |
| **CKD 4–5 期 (eGFR < 30)** | 0.6–0.8（医监） | 36–48g | 42–56g |

> **换算常识**：1个鸡蛋 ≈ 6–7g 蛋白；100g 鸡胸肉 ≈ 23–25g；100g 豆腐 ≈ 8g；250ml 牛奶 ≈ 8g

### C3. 优质蛋白来源排序

| 排名 | 食物 | 每100g 蛋白质 | 亮氨酸含量 | 适用性 |
|------|------|-------------|-----------|--------|
| 🥇 | **鸡蛋（全蛋）** | ~13g | 高（1.1g/个） | 易消化、烹饪简单、咀嚼容易 |
| 🥇 | **鸡胸肉/瘦牛肉** | ~23–25g | 高（2.0–2.5g/100g） | 需切小块/剁碎，60+ 牙齿问题需注意 |
| 🥇 | **鱼类（三文鱼/鲈鱼）** | ~20–22g | 中高 | Omega-3 协同抗炎，骨刺需处理 |
| 🥈 | **牛奶/酸奶（无糖）** | ~3–3.5g/100ml | 中 | 乳糖不耐患者可选无乳糖/酸奶 |
| 🥈 | **豆腐/豆制品** | ~8g（豆腐）/ 15g（千张） | 中 | **CKD 患者优选！** 植物蛋白对肾负担小 |
| 🥉 | **大豆/毛豆** | ~12–13g | 中 | 富含异黄酮，60+ 女性尤其适合 |
| 🥉 | **虾/海鲜** | ~18–20g | 中高 | 低脂高蛋白，注意胆固醇（建议 ≤ 300mg/天） |

### C4. 精制碳水识别清单

> "减精碳" = 减少**精制碳水（refined carbohydrates）**
> 保留**复合碳水（complex carbohydrates）** 作为能量来源

| 🚫 限制/避免 | ✅ 替换建议 | 备注 |
|-------------|-----------|------|
| **含糖饮料**（可乐/果汁/奶茶） | 白开水/淡茶/无糖柠檬水 | 一杯 500ml 可乐 ≈ 54g 精制糖（WHO 建议 ≤ 25g/天） |
| **白米饭** | 糙米/燕麦/杂粮饭（杂粮占 1/3–1/2） | 白米 GI ≈ 73；糙米 GI ≈ 55 |
| **白面（馒头/面条/白吐司）** | 全麦/荞麦/燕麦/山药/红薯 | 白馒头 GI ≈ 85；全麦面包 GI ≈ 50–60 |
| **甜点/蛋糕/饼干** | 水果（非烘焙）/ 坚果（无糖） | 一块蛋糕 ≈ 20–30g 精制糖 |
| **含糖麦片/早餐糊** | 纯燕麦片（无添加糖） | 市售"营养麦片"常见含糖量 20–30g/100g |
| **过加工红薯/芋头（拔丝/油炸）** | 蒸/煮/烤 | 油炸破坏膳食纤维 |

### C5. 三餐建议结构（示例，**75kg 男性**，每日 90g 蛋白目标）

| 餐次 | 食物组合 | 蛋白质(g) | 精制碳水控制 |
|------|---------|-----------|------------|
| **早餐 7:30** | 鸡蛋 2 个（煮）+ 牛奶 250ml + 全麦面包 1 片 + 一小把核桃 | ~22g | ✅ 全麦替代白面包 |
| **午餐 12:00** | 鸡胸肉 100g + 杂粮饭 200g + 蒸西兰花/胡萝卜 + 豆腐汤 | ~40g | ✅ 杂粮替代白米 |
| **加餐 15:30** | 无糖酸奶 200g + 一小把杏仁 + 蓝莓 | ~10g | ✅ 避开含糖饮料/饼干 |
| **晚餐 18:00** | 清蒸鱼 100g + 糙米饭 100g + 焯青菜 + 玉米 半根 | ~30g | ✅ 糙米代替白米 |
| **总计** | | **~102g ✅ 达到 1.36g/kg 水平** | 复合碳水占比 > 80% |

> 注：原示例"60kg 女性 / 72g 目标"经核算蛋白偏低（仅 94g/1.57g/kg）超出健康范围, 已修正为 75kg 男性 / 90g 目标 / ~102g (1.36g/kg 水平) — 此条目作为示例参考, 用户实际应按自身体重×1.0-1.5g/kg 估算。

### C6. ⏱ 运动与蛋白质配合时机

- **黄金窗口**：运动结束后 **30 分钟内**补充蛋白质（肌肉蛋白合成 MPS 峰值在此窗口）
- **60+ 特殊性**：中老年肌肉合成代谢对氨基酸的敏感度下降约 **30%**（"anabolic resistance"），因此：
  - 运动后补蛋白比年轻人**更关键**
  - 推荐剂量：**20–30g 优质蛋白**（≈ 3–4 个鸡蛋 / 或 25g 乳清蛋白粉）
  - 亮氨酸含量 ≥ **2.5g** 的蛋白来源（乳清/鸡蛋/大豆分离蛋白均符合）
- **实际建议给用户的文案**：
  > "练完后半小时内喝一杯牛奶 + 吃两个鸡蛋，效果最好。别超过 1 小时。"
- **参考文献**：
  - Moore DR et al. (2015) — protein timing in older adults
  - Areta JL et al. (2013) — leucine threshold for MPS
