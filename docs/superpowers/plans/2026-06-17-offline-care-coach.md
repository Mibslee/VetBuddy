# Offline Care Coach Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the offline "today coach" experience that makes VetBuddy more direct, supportive, and easier for older users.

**Architecture:** Add small local stores and settings wrappers, then connect them to existing SwiftUI views. Keep persistence in UserDefaults and existing diet stores; avoid networking, accounts, and online generation.

**Tech Stack:** Swift, SwiftUI, UserDefaults, AVFoundation, Xcode project generated by XcodeGen.
